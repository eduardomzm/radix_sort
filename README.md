Radix Sort
Otra forma de ordenar números, pero sin compararlos entre sí.
Es como ordenar documentos por fecha. Primero agrupa los números
según su último dígito, luego los vuelve a agrupar por el siguiente,
y así sucesivamente. Al terminar con todos los dígitos, la lista queda ordenada.
Sirve para istas de números enteros donde no queremos hacer comparaciones complejas
